ABOUT

We are a team of computer programmers who believe that anybody can develop code with the right mindset. Our program will help students learn how to develop and build code with user-friendly tools to create different multi-functional apps.

TOOLS

In order to create apps on the fly, we believe that MIT App Inventor is the best user-friendly software to create Android apps. MIT App Inventor features basic drag and drop user interface designing and block coding. Our team also is capable of teaching simple Java concepts and Microsoft Makecode Arcade.

SCHEDULE

Our program offers a series of 1.5 hour lecture/workshop sessions where students will be guided through exercises using MIT's popular App Inventor. These lessons will allows the students to build apps and focus on hands-on learning, with very little to no lecturing. This program will be held on consecutive Fridays from February 22nd to May 3rd.
